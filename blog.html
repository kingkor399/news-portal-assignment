Another difference between var and let/const is that var variables are function-scoped, while let and const variables are block-scoped. This means that var variables are only available within the function they were declared in. Arguments objects are not available in arrow functions, but are available in regular functions. Regular functions created using function declarations or expressions are ‘constructible’ and ‘callable’. Since regular functions are constructible, they can be called using the ‘new’ keyword.

The first difference between map () and forEach () is the returning value. The forEach () method returns undefined and map () returns a new array with the transformed elements. Even if they do the same job, the returning value remains different.
